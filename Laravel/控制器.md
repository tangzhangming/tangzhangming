## Laravel控制器
```PHP
// 默认两个trait
class Controller extends BaseController
{
    use AuthorizesRequests, ValidatesRequests;
}
```

## AuthorizesRequests
```PHP
//授权当前用户的给定操作
authorize($ability, $arguments = [])

//授权指定用户的给定操作
authorizeForUser($user, $ability, $arguments = [])

//如果未提供，则猜测能力的名称。
parseAbilityAndArguments($ability, $arguments)
```
